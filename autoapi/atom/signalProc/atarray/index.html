calcPathInfo()#

Calculate path vectors, lengths, and orientations based on microphone and speaker locations.

The method calculates: - Path vectors: Vector from speaker to microphone. - Path lengths: Distance from speaker to microphone. - Path orientations: Angular orientation of the path.

setupPathIntegrals()#

Sets up path integral calculations for a given set of speakers and microphones.

Notes

This method does not return any value, but modifies the class instance by adding a new attribute ‘self.ds’ which is an xarray Dataset containing the information about the paths, points for integrations, and Euclidean distances between successive points.

It expects that the following attributes are pre-defined in the instance: ‘atarray.ds.speakerLocations’, ‘atarray.ds.micLocations’.

The total number of points for integration along each path is hard-coded as 250.
