class atom.signalProc.audiodata.AudioData#

Class to handle audio data. Includes attributes related to the audio signals and methods for loading data, checking data validity, extracting reference signals, and exporting/importing data.

samplingFrequency#

The frequency at which the audio data was sampled, default is 20000 Hz.

Type:

float

deltaT#

Time increment value, default is 0.05 seconds.

Type:

float

recordTimeDuration#

The duration of the audio recording, default is 0.5 seconds.

Type:

float

recordTimeDelta#

The time difference between recordings, default is 0.00005 seconds.

Type:

float

recordLength#

The length of the audio record, default is 10000.

Type:

int

nFrames#

The number of frames in the audio data, default is 120.

Type:

int

chirpTimeDuration#

The duration of the chirp signal, default is 0.0058 seconds.

Type:

float

chirpRecordLength#

The length of the chirp record, default is 116 samples.

Type:

int

chirpCentralFrequency#

The central frequency of the chirp, default is 1200 Hz.

Type:

float

chirpBandwidth#

The bandwidth of the chirp, default is 700 Hz.

Type:

float

speakerSignalEmissionTime#

Array of speaker signal emission times.

Type:

np.ndarray

nMics#

The number of microphones, default is 8.

Type:

int

nSpeakers#

The number of speakers, default is 8.

Type:

int

windowHalfWidth#

The half-width of the window, default is 0.01 seconds.

Type:

float

__post_init__()#
loadData(dataPath, keepSpkData=False)#

Load the main data from the specified path.

Parameters:

  • dataPath (str) – Path to microphone and speaker data.

  • keepSpkData (bool, optional) – Flag to indicate whether the full record of speaker data should be kept, or just a reference signal. Default is False.

_checkData(data, nChannels)#

Private method to sanity check data size/shape against config inputs.

Parameters:

  • data (numpy.ndarray) – Data to be checked.

  • nChannels (int) – Number of channels as per config.

getReferenceSignal(spkData)#

Extract a reference signal from one speaker.

Parameters:

spkData (numpy.ndarray) – Speaker data from which the reference signal needs to be extracted.
